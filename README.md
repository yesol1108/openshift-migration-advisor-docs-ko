## GitHub Pages 설정

Repository의 **Settings → Pages**에서 Source를 **Deploy from a branch**, Branch를 **main**, folder를 **/(root)**로 설정합니다.

이 저장소는 `.nojekyll`을 사용하여 GitHub Pages의 Jekyll 자동 렌더링을 끄고, 정적 HTML 파일을 그대로 서빙하도록 구성했습니다.
