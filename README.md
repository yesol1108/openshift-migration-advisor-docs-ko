## GitHub Pages 활성화 방법

1. Repository의 **Settings**로 이동합니다.
2. 왼쪽 메뉴에서 **Pages**를 선택합니다.
3. **Build and deployment**에서 Source를 **Deploy from a branch**로 선택합니다.
4. Branch를 **main**, folder를 **/(root)**로 선택하고 저장합니다.
5. 몇 분 후 `https://yesol1108.github.io/openshift-migration-advisor-docs-ko/` 주소로 접속합니다.
