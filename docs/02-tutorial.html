튜토리얼: Migration Assessment 사용 방법

VMware 환경을 OpenShift Virtualization으로 이전하기 전에 평가하는 대표적인 방식은 RVTools Flow와 Discovery Agent Flow입니다.

RVTools Flow

기존 RVTools export Excel 파일을 업로드하여 즉시 assessment를 수행합니다. vCenter 접근이 어렵거나 초기 검토가 필요한 경우에 적합합니다.

필수 요구사항

  • vInfo 시트에 최소 1개 행이 있어야 합니다.
  • VM ID, VM 또는 Name, CLUSTER, VI SDK UUID 컬럼이 필요합니다.

권장 시트

vHost, vDatastore, vNetwork, vCPU, vMemory, vDisk, vNic 시트가 있으면 더 완전한 평가가 가능합니다.

Discovery Agent Flow

Discovery Agent를 vCenter에 OVA appliance로 배포하여 live inventory를 수집합니다. vCenter API에서 직접 데이터를 수집하므로 더 정확한 평가에 유리합니다.

선택 기준

방식장점적합한 상황
RVTools Flow빠르고 간단함초기 진단, vCenter 접근 제한
Discovery Agent Flow정확한 live inventory구체적인 migration planning
