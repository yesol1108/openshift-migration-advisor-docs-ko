Migration Advisor

OpenShift Migration Advisor는 VMware 환경을 Red Hat OpenShift Virtualization으로 이전하려는 사용자가 사전에 환경을 평가할 수 있도록 돕는 assessment 도구입니다.

주요 기능

  • VMware vCenter 환경 분석
  • VM 및 VM 구성 정보 식별
  • 마이그레이션 준비 상태 평가 제공
  • 인프라, 스토리지, 네트워크 관련 리포트 생성

환경 평가 방법

RVTools Upload

이미 VMware 환경에 대한 RVTools export 파일이 있다면 Excel 파일을 업로드하여 빠르게 평가를 시작할 수 있습니다.

Discovery Agent

경량 OVA appliance를 vCenter에 배포하여 라이브 환경을 탐색합니다. Agent는 VMware 환경에서 직접 상세 정보를 수집하고 실시간 평가 데이터를 제공합니다.

핵심 가치

  • VM, Host, Datastore, Network, Cluster 정보를 기반으로 migration readiness를 확인합니다.
  • 마이그레이션 전에 blocking issue, warning, sizing, 예상 시간을 검토할 수 있습니다.
  • OpenShift Virtualization migration planning의 초기 근거 자료로 활용할 수 있습니다.
이 문서는 원문 문서를 기반으로 한 한국어 번역·정리본입니다.
