Migration Forecaster

Migration Forecaster는 VMware datastore pair 간 실제 disk-copy benchmark를 수행하여 storage throughput과 migration time을 예측합니다.

동작 방식

source datastore와 target datastore 사이에서 benchmark disk copy를 여러 번 반복하고 평균, 중앙값, 표준편차, 신뢰구간을 계산합니다.

생성되는 임시 리소스

  • Filler VM: Alpine Linux를 부팅해 benchmark disk를 random data로 채웁니다.
  • Benchmark disk: source datastore에 생성되는 기본 10GB thin VMDK입니다.
  • Clone disk: 각 iteration마다 target datastore에 생성되는 copy입니다.
  • Seed ISO와 임시 directory도 생성되며 benchmark 종료 후 정리됩니다.

Filler VM 사양

  • 1 vCPU, 256MB RAM
  • Alpine Linux 기반
  • 네트워크 연결 불필요
  • disk fill 완료 후 자동 power off

필요 권한

Datastore 파일 생성/조회/삭제/업로드 권한과 VM 생성/삭제/clone/power-on/device attach 관련 권한이 필요합니다.

API 흐름

  1. Credential 검증
  2. Datastore 목록 조회
  3. Pair capability 확인
  4. Benchmark 시작
  5. Status polling
  6. Run data 및 통계 확인

결과 해석

결과에는 mean/median/min/max throughput, 표준편차, 95% 신뢰구간, 1TB 기준 best/expected/worst case estimate가 포함됩니다.

이 문서는 원문 문서를 기반으로 한 한국어 번역·정리본입니다.
