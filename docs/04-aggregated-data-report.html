Aggregated Data Report: 수집 데이터와 의미

Aggregated Data Report는 Migration Advisor가 수집한 VMware inventory data를 마이그레이션 계획에 활용할 수 있도록 요약한 리포트입니다.

  • 적용 버전: V0.4.0
  • 마지막 업데이트: 02-02-2026

Virtual Machine Data

VM section에는 전체 VM 수, migration 가능한 VM 수, warning이 있는 VM 수, migration 불가 VM 수 등이 포함됩니다.

Resource Breakdown

  • CPU cores: 전체 및 상태별 vCPU core 수
  • RAM: 전체 및 상태별 메모리 용량
  • Disk storage/count: 전체 disk 용량과 disk 개수

Distribution Metrics

CPU tier, memory tier, NIC count, disk size tier 등으로 VM이 어떻게 분포되어 있는지 확인할 수 있습니다.

Operating System Information

각 detected OS별 VM 수, MTV migration 지원 여부, upgrade recommendation을 확인할 수 있습니다. unsupported OS는 반드시 migration 실패를 의미하는 것이 아니라 Red Hat의 공식 검증/지원 범위 밖이라는 의미에 가깝습니다.

Issue / Warning

리포트에는 migration을 차단할 수 있는 issue와 사전 검토가 필요한 warning이 함께 표시됩니다. 이 정보는 migration scope와 사전 조치 항목을 정리하는 데 사용합니다.

이 문서는 원문 문서를 기반으로 한 한국어 번역·정리본입니다.
